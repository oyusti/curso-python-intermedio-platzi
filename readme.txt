En este carpeta utilizaremos python en un ambiente virtual "venv" propio de python
Asimismo utilizaremos Git en este proyecto

Para crear el entorno virtual utilizaremos
python -m venv venv

Donde el primer "venv" es la palabra reservada para crear el entorno y la segunda "venv" es el nombre del entorno que puede ser el que tu quieras

Para activar el entorno utilizamos lo siguiente
source venv/bin/activate.

Notaras que esta activo porque en la parte izquierda aparece la palabra"(venv)"

y para desactivarlo utilizamos 
Deactivate
